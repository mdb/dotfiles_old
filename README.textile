h1. About 

Basic backup of some dotfiles and and utilities.

h1. Installation

*Copy these files to your home directory:*

* .bashrc
* .bash_profile
* .vimrc
* .gvimrc

*Copy these files to your .subversion directory:*

* misc/config
* misc/svn-editor.script (Used for auto commit message formatting)
* misc/svn-editor.sh (Used for auto commit message formatting)
